🔒 นโยบายความเป็นส่วนตัว

อัปเดตล่าสุด: 13 มิถุนายน 2569

CheckBank พัฒนาโดย OHM2 (sparkone.dev) นโยบายนี้อธิบายวิธีที่เราเก็บรวบรวม ใช้ และปกป้องข้อมูลของคุณ

1. ข้อมูลที่เราเก็บ

  • ข้อมูลบัญชี: อีเมล รหัสผ่าน (เข้ารหัสโดย Supabase Auth)
  • ข้อมูลการเงิน: รายรับ-รายจ่าย บัญชีธนาคาร เป้าหมายการออม ที่คุณบันทึกเอง
  • รูปภาพสลิป: ส่งไปประมวลผลที่ Groq API และไม่ถูกเก็บไว้หลังประมวลผล
  • ข้อมูลโปรไฟล์: ชื่อที่แสดง และตัวการ์ตูนที่เลือก

2. วิธีที่เราใช้ข้อมูล

  • แสดงยอดเงิน สถิติ และสรุปการเงินของคุณ
  • ประมวลผลสลิปด้วย AI เพื่ออ่านข้อมูลอัตโนมัติ
  • บันทึกเป้าหมายการออมและติดตามความคืบหน้า

3. การแชร์ข้อมูลกับบุคคลที่สาม

  • Supabase: ฐานข้อมูลและระบบ Auth
  • Groq API: ประมวลผลรูปภาพสลิปด้วย AI ผ่าน Edge Function ของเรา ไม่ถูกเก็บถาวร
  • Google OAuth: สำหรับผู้ที่เลือกเข้าสู่ระบบด้วย Google

เราไม่ขายข้อมูลของคุณให้บุคคลใดทั้งสิ้น

4. ความปลอดภัย

ข้อมูลทั้งหมดถูกเข้ารหัสระหว่างส่ง (TLS) และเข้าถึงได้เฉพาะบัญชีของคุณผ่าน Row Level Security ของ Supabase

5. สิทธิ์ของคุณ

  • ดู แก้ไข หรือลบข้อมูลของคุณได้ตลอดเวลาในแอป
  • ลบบัญชีและข้อมูลทั้งหมดได้ที่: ตั้งค่า → ลบบัญชี

6. การลบบัญชีและข้อมูล

วิธีลบบัญชี:
เปิดแอป → ตั้งค่า → เลื่อนลงล่าง → กด "ลบบัญชี" → ยืนยัน 2 ครั้ง

การลบบัญชีจะลบข้อมูลทั้งหมดอย่างถาวร ได้แก่ ธุรกรรม บัญชี เป้าหมาย และสลิปทั้งหมด

7. นโยบายเกี่ยวกับเด็ก

แอปนี้ไม่ได้ออกแบบสำหรับผู้ที่มีอายุต่ำกว่า 13 ปี
